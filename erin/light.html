Light em' All!

For my final project in my Fundamentals of Computer Science II class, we made a mini game that scrambles a bunch of pipes, and the user has to connect all the pipes and move a power source to the right location.

This game was made using Java and it's world library. It uses Kruskal's Algorithm to generate a solvable random board. The algorithm uses a minimum spanning tree, sorted with a heapsort.

If you want to know more, please message me! The code is currently private due to class policies.
