Don't lose your snooze!



Don't lose your snooze was a four person project for a final in my Programming Basics class. The idea revolves around sleep, where the person is able to input all sorts of data and see trends. Did you drink 5 cups of coffee and found that your sleep quality was only 20%? Maybe you should drink less coffee. How about discovering that you sleep best when you work for exactly 7 hours? We wanted our app to extend to allow the user to input whatever they wanted to document.



Throughout the entire project, while this was supposed to be a four person project, I did almost everything to the current state of the website. Setting up a repository for the project, extracting data from firebase, allowing the user to push data on to the server, and displaying all the data on our chart. The only thing I didn't do was the design, and I'm grateful that at least one of my partners dealt with the visuals.
