Light Em' All!

A game where you light all the pipes and move a powersource.

Light Em' All is a minigame that scrambles all the pipes, and the user has to connect all the pipes so that everything is connected. Once that is accomplished, the user then has to move the "powersource" to the right location. Also, this was the final project of my Fundamentals of Computer Science 2 course.

This game was made using Java and it's world library. All the visuals were made using the world library. However, there was a lot of thought putting into this to make it work correctly.
